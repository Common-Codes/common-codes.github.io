Projects by the Common-Codes Team

Shnitters

Formerly: ShnittersPWA

Shnitters is an Open Source Text sharing platform, where you create publications known as "posts".

Has an account flow, usernames and handles, Account Webhooks (Payload URL's for sharing posts to other socials) and a smooth "Twitter" UI/UX.

Uses URL Search queries. Heavy JS dependency.

Shnitters has self-hosting options.


uTube

Formerly: OpenSource-uTube

uTube is a multimedia sharing platform, designed for video content sharing specifically. Most videos are memes or meme dumps. "Channel"'s are UUID specified.

Uses slower URL Hash queries. Heavy JS dependency.

uTube has self-hosting options.


Verbose

Lightweight Portable (and pluggable) Image Proxy and Embed Generator. It is designed to be ultra-lightweight as well as slick and fast.

Uses a combination of URL Search queries and slower URL Hash queries. Heavy JS dependency.

Verbose is designed for self-hosting.


Crispy-Chat

Formerly: Crispy

An Open Source Chatting Website* powered by Firebase, EJS and some internet magic ;p

*Implementation of jankyCord

Crispy has self-hosting options available!


Fire

Multimedia (Target's Music (audio)) sharing platform.

The API allows you to play music on Discord from the app :o
